## ⚙️ Technical Specs for All Uploads

| Property | Value |
|----------|-------|
| Accepted Formats | `.jpg` `.jpeg` `.png` `.pdf` |
| Maximum File Size | 2MB per file |
| Upload Method | HTML file input (client-side) |
| Upload Trigger | `onchange` event |
| Validation | Presence check only (no size/format validation in code) |
| Success Indicator | Zone border turns **green**, icon changes to ✅ |
| Failure Indicator | Zone border turns **red** for 2.5 seconds |
| File Name Display | Shown inside upload zone after selection |
| Storage Variable | `uploadedFiles{}` object (key = doc ID, value = filename) |
